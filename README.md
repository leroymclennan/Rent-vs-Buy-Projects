# Rent-vs-Buy-Projects
A collection of projects in python and Google Sheets/Excel aiming to investigate long term investment returns for a variety of strategies, with the goal of answering "Should I rent or buy given my current finances?".

In its current stage, the attached spreasheet explores the 30 year investment returns associated with 4 different "rent vs buy" strategies. These strategies include:
- Renting and investing surplus cashflow in a generic, broad market ETF
- Buying a Primary Place of Residence (PPOR), paying the minimum deposit and contributing no extra to mortgage repayments, and investing surplus cashflow into a generic, broad market ETF
- Buying a Primary Place of Residence (PPOR) using all available funds as a deposit, then withdrawing the maximum from the property equity to set up an investment loan, and using that loan to invest in a generic, broad market ETF, such that the investment loan interest repayments are tax deductible. Any surplus cahsflow is then also invested into the generic, broad market ETF.
- Buying an Investment Property (IP) to maximise negative gearing, whilst renting, and investing all surplus cahsflow into a generic, broad market ETF

Future aims for this project are to create a python program that presents the same information, however that also contains an elegant and efficient user interface for inputting data related to an individual's starting financial position, and the characteristics of the property that they are considering renting and/or buying. This project has been delayed though, as I don't see it offering any superior visualisations or conclusions compared to the spreadsheet in its current form; the python project would serve no purpose than to demonstrate my python coding abilities.
